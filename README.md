# 合同会社変更登記 書類自動生成ツール

アンケート形式で情報を入力すると、以下の3書類が自動生成・印刷できるWebツールです。

- 同意書（総社員の同意書）
- 就任承諾書
- 辞任届

## 機能

- 入力フォームにリアルタイム連動する書類プレビュー
- 3書類の一括印刷（ブラウザの印刷機能）
- 入力データのJSON保存・読込（案件の再利用）
- サンプル入力機能（動作確認用）

## 使い方

1. ブラウザでこのサイトにアクセス
2. フォームに必要情報を入力
3. プレビューを確認
4. 「3書類すべてを印刷」ボタンで印刷
